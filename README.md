# VillageIbook bootstraped with Next.js, ChakraUI, Redux Toolkit, TypeScript
## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deploy on Vercel
